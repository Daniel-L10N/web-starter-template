# 🚀 Web Starter Template

A modern, production-ready starter template for web projects.

## Features

- ⚡ **Fast** - Built for performance
- 📦 **TypeScript** - Type-safe code
- 🎨 **Styled** - CSS Modules ready
- 🧪 **Testing** - Jest configured
- 📝 **Linting** - ESLint + Prettier

## Quick Start

```bash
# Use as template
npx create-next-app my-app --template https://github.com/Daniel-L10N/web-starter-template

# Or clone directly
git clone https://github.com/Daniel-L10N/web-starter-template.git my-project
cd my-project
npm install
npm run dev
```

## Structure

```
├── src/
│   ├── components/   # React components
│   ├── pages/         # Page routes
│   ├── styles/        # CSS modules
│   └── utils/         # Utility functions
├── public/            # Static assets
├── package.json
└── README.md
```

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
